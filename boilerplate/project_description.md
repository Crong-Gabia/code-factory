# Project Description

## Purpose

- 이 프로젝트는 컨플루언스나 노션등과 같은 공동 편집이 가능한 형태의 지식/문서 관리 서비스의 백엔드 API 프로젝트입니다.
- 문서는 Markdown을 기반으로 작성하며, 문서 간 참고 링크가 가능하도록 설계되어 각 문서는 고유한 문서 URL을 가지게 됩니다.
- 공개 범위는 공간, 페이지 트리, 개별 페이지에서 각각 지정이 가능합니다.

## Scope

- In scope:
- 공간/페이지 문서 CRUD API
- 문서 조회/목록 API
- 문서 간 링크(참조) 처리
- Out of scope:
